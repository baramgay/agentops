## Reddit (r/LocalLLaMA, r/MachineLearning)

**Show HN: agentops — structured multi-agent workflows using plain markdown files**

Instead of using a complex orchestration framework, I built a system where each AI agent is defined by a `role.md` and `memory.md` file. A simple Python script manages status updates and a local dashboard tracks everything in real time.

The vertical command hierarchy (user → orchestrator → team lead → agent) ensures nothing gets lost and every action is auditable.

Repo: github.com/baramgay/agent
