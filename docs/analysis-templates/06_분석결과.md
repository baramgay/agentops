# [6] Ⅳ 분석결과 — 상세 규칙

> 보고서의 핵심 본문. 분석 결과를 중섹션·소섹션 단위로 전개. 가변 페이지(10~25p).

---

## 대섹션 제목

| 항목 | 규칙 |
|------|------|
| 표기 | `Ⅳ` + 줄바꿈 + `분석결과` (또는 `Ⅳ분석결과` 한 줄) |
| 폰트 | Bold, 16~18pt |
| 위치 | 새 페이지 시작 |
| 서식 | Ⅰ~Ⅲ과 동일한 컬러바/배경 스타일 |

> 세 보고서 모두 Ⅳ는 새 페이지에서 시작. 직전 Ⅲ 프로세스와 같은 페이지에 이어쓰지 않음.

---

## 제목 체계 (3단계)

```
Ⅳ                                         ← 대섹션 (16~18pt, 컬러바)
분석결과

1  고령 의료 수요·공급 현황                   ← 중섹션 (12~14pt, 배지)
  가  시군별 고령 생활인구 현황               ← 소섹션 (11pt, Bold)
  • 불릿 ...
  [그림] 시군별 고령 생활인구 현황

  나  의료기관 공급 현황
  • 불릿 ...
  [그림] 시군별 의료기관 공급 현황

2  취약지역 종합 현황
  가  취약점수 분포
  ...
```

### 중섹션 번호 체계

| 항목 | 규칙 |
|------|------|
| 번호 | 아라비아 숫자 1, 2, 3, 4, 5, 6... |
| 스타일 | 원형 배지(흰색 숫자+컬러 원) 또는 사각 배지 |
| 폰트 | Bold, 12~14pt |
| 간격 | 위: 새 페이지 또는 이전 소섹션 후 16pt |

### 소섹션 번호 체계

| 항목 | 규칙 |
|------|------|
| 번호 | 한글 가, 나, 다, 라, 마... |
| 폰트 | Bold, 11pt |
| 들여쓰기 | 중섹션 기준 좌측 0.5~1cm |
| 간격 | 위: 12pt, 아래: 4pt |

## 페이지 단위 구성 패턴

### 패턴 A: 기본형 (가장 빈번)

```
┌─────────────────────────────────┐
│ 가  [소섹션 제목]                │
│ •  불릿 1 (관측 사실)            │
│ •  불릿 2 (관측 사실)            │
│ •  불릿 3 (비교/대조)            │
│ •  불릿 4 (해석)                │
│ •  불릿 5 (해석)                │
│ •  불릿 6 (정책적 함의)          │
│                                 │
│ ┌─────────────────────────┐     │
│ │       [차트 이미지]       │     │
│ └─────────────────────────┘     │
│ [그림] 캡션 제목                 │
└─────────────────────────────────┘
```

**밀도:** 불릿 5~6개 + [그림] 1개 = 1p

### 패턴 B: 차트 2개형 (차트 비교 페이지)

```
┌─────────────────────────────────┐
│ ┌────────────┐ ┌────────────┐   │
│ │  [차트 1]   │ │  [차트 2]   │   │
│ └────────────┘ └────────────┘   │
│ [그림] 캡션 1                    │
│ [그림] 캡션 2                    │
└─────────────────────────────────┘
```

**밀도:** [그림] 2개 + 불릿 0~2개 = 1p
- 불릿 해석 생략 또는 최소화
- 차트가 자명한 비교 목적일 때 사용

### 패턴 C: 표 중심형

```
┌─────────────────────────────────┐
│ 가  [소섹션 제목]                │
│ •  불릿 1~3개 (표 해석)          │
│                                 │
│ ┌─────────────────────────┐     │
│ │  순번│유형│시군│행정동│점수│  │     │
│ │  1  │B  │...│...  │48.4│  │     │
│ │  2  │B  │...│...  │47.5│  │     │
│ │  ... (10~20행)          │     │
│ └─────────────────────────┘     │
│ [표] 캡션 제목                   │
└─────────────────────────────────┘
```

**밀도:** 불릿 1~3개 + [표] 1개 = 1p
- 표 행이 15개 이상이면 2p로 분할

### 패턴 D: 지도 전체형 (단계구분도)

```
┌─────────────────────────────────┐
│ •  불릿 4~6개 (지도 해석)        │
│                                 │
│ ┌─────────────────────────┐     │
│ │                         │     │
│ │    [경남 지도 이미지]     │     │
│ │                         │     │
│ └─────────────────────────┘     │
│ [그림] 캡션 제목                 │
└─────────────────────────────────┘
```

- 지도는 페이지 폭의 80~90% 차지
- 불릿은 지도 위에 배치

## 불릿 해석 작성 규칙 (상세)

### 6불릿 표준 구조

| 순서 | 역할 | 예시 시작부 |
|------|------|------------|
| 1번 | 가장 두드러진 관측 | "~ 기준으로는 ~가 가장 ~" |
| 2번 | 후속 관측/분포 | "~은 ~로 ~가 뒤를 잇는 구조가 확인됨" |
| 3번 | 비교/대조 | "~는 ~보다 ~으로 ~과는 차이를 보임" |
| 4번 | 내부 편차/예외 | "같은 ~ 안에서도 ~의 차이가 ~" |
| 5번 | 해석/인과 | "이 결과는 ~을 보여줌", "~로 해석 가능함" |
| 6번 | 활용/함의 | "~ 시 ~으로 활용 가능", "~을 함께 봐야 함을 보여줌" |

### 종결어미 사전

| 종결어미 | 용도 | 빈도 |
|----------|------|------|
| ~으로 나타남 | 관측 결과 기술 | 매우 높음 |
| ~이 확인됨 | 사실 확인 | 높음 |
| ~을 보여줌 | 시사점 도출 | 높음 |
| ~을 시사함 | 해석/추론 | 중간 |
| ~이 큼 / ~이 적음 | 비교 결과 | 중간 |
| ~할 수 있음 | 활용 가능성 | 중간 |
| ~할 필요가 있음 | 제언 | 낮음 (마지막 불릿) |

### 수치 표기 규칙

```
✅ 올바른 예시:
  "김해시가 8개로 가장 많고 밀양시 6개, 거제시와 남해군이 각 5개로 뒤를 잇음"
  "의령군 85%, 함양군 82%, 고성군 79% 순으로 높게 나타남"
  "양산시 물금읍이며 취약점수는 100.0으로 확인"
  "2018년 842,805명에서 2025년 639,586명으로 203,219명(-24.1%) 감소"

❌ 잘못된 예시:
  "여러 시군에서 높게 나타남"           ← 수치 없음
  "상당히 많은 지역이 취약함"           ← 모호한 표현
```

- 매 불릿에 수치 1개 이상 반드시 포함
- 순위 나열: 최대 5개까지 (1위, 2위, ..., 5위)
- 증감: 절대값 + 비율 병기 (예: "203,219명(-24.1%)")
- 소수점: 소수 1자리까지 (예: "17.9km", "74.6%")

## [그림] 캡션 작성 규칙 (상세)

### 캡션 패턴

```
[그림] {공간단위}별 {지표명} {차트유형/수식어}

예시:
[그림] 시군별 고령 생활인구 현황
[그림] 행정동별 고령 생활인구 현황 지도
[그림] 취약점수 분포
[그림] 취약점수 단계구분도
[그림] 수요·공급 산포도 — 유형 구분
[그림] 시군별 취약 행정동 비율 롤리팝 차트
[그림] 소득-순유입 버블 차트 (버블 크기=생활인구)
[그림] 시군별 청년 1인당 카드 이용건수
[그림] 경남 청년(20~39세) 연도별 등록인구 추이 (2018~2025)
```

### 캡션 구성 요소

| 요소 | 필수 | 예시 |
|------|------|------|
| 공간 단위 | 권장 | "시군별", "행정동별", "경남" |
| 대상 집단 | 선택 | "고령", "청년(20~39세)" |
| 지표명 | 필수 | "생활인구", "취약점수", "통근거리" |
| 차트 유형 | 선택 | "지도", "단계구분도", "산포도", "롤리팝 차트" |
| 기간 | 선택 | "(2018~2025)", "(2025년)" |
| 부가 설명 | 선택 | "— 유형 구분", "(버블 크기=생활인구)" |

### 캡션 종결 패턴

```
~현황        : 막대 차트, 기본 비교
~분포        : 히스토그램, 분포도
~추이        : 선 그래프, 시계열
~지도        : 단계구분도, 공간 시각화
~비교        : 그룹별 비교 차트
~관계        : 산포도, 상관분석
```

## [표] 서식 규칙 (상세)

### 표 구조

```
┌────────┬────────┬────────┬────────┬────────┐
│ 시군구 │취약비율│ 1등급 수│   A    │   B    │  ← 헤더 (배경색, Bold)
├────────┼────────┼────────┼────────┼────────┤
│창원시  │ 57.1   │   4    │   4    │   0    │  ← 데이터 행
│남해군  │ 50.0   │   5    │   0    │   5    │
│...     │        │        │        │        │
└────────┴────────┴────────┴────────┴────────┘
[표] 시군별 취약비율 상위 현황                     ← 캡션 (표 아래)
```

### 정렬 규칙

| 컬럼 유형 | 정렬 | 예시 |
|-----------|------|------|
| 지역명/텍스트 | 좌측 정렬 | 창원시의창구, B유형 |
| 숫자(정수) | 우측 정렬 | 4, 8, 15 |
| 숫자(소수) | 우측 정렬, 소수점 정렬 | 57.1, 100.0 |
| 비율(%) | 우측 정렬 | 85%, -24.1% |

### 단위 표기

```
✅ 헤더에 단위 포함: "평균통근거리(km)", "통근인구수(명)", "월소득(만원)"
❌ 데이터 셀에 단위: "23.1km" → "23.1" (단위는 헤더로)
```

### 행 수 가이드

| 표 유형 | 행 수 | 페이지 |
|---------|-------|--------|
| 요약 표 (상위 5개) | 5~7행 | 0.3p |
| 시군 전체 표 | 18행 | 0.5~0.7p |
| 상세 목록 (Top 20) | 17~20행 | 0.7~1p |

## 중섹션 구성 패턴 (세 보고서 기반)

### 의료접근성 (Ⅳ 분석결과: 중섹션 6개)

```
1  고령 의료 수요·공급 현황          (소섹션 3개: 가~다, 7p)
2  취약지역 종합 현황               (소섹션 3개: 가~다, 5p)
3  취약 유형 분류                  (소섹션 2개: 가~나, 2p)
4  수요·공급 분석                  (소섹션 2개: 가~나, 2p)
5  위치 기반 의료접근성 분석         (소섹션 3개: 가~다, 3p)
6  우선 검토 후보지                (소섹션 2개: 가~나, 3p)
```

### 청년정착 (Ⅳ 분석결과: 중섹션 6개)

```
1  경남 청년 인구 현황              (소섹션 2개: 가~나, 5p)
2  청년 인구 유출입 구조            (소섹션 3개: 가~다, 5p)
3  청년 생활인구 및 유동인구 특성    (소섹션 3개: 가~다, 7p)
4  청년 소비 활력 분석 (KB카드)     (소섹션 2개: 가~나, 3p)
5  청년 소득·경제 여건 분석 (KCB)   (소섹션 3개: 가~다, 5p)
6  청년 정착잠재지수 종합 분석       (소섹션 2개: 가~나, 4p)
```

### 통근실태 (Ⅳ 분석결과: 중섹션 5개 + 시군별 18p)

```
1  경남 통근 현황 개괄              (소섹션 3개: 가~다, 3p)
2  통근 패턴의 공간적 특성          (소섹션 2개: 가~나, 2p)
3  소득 수준과 통근 패턴의 관계      (소섹션 3개: 가~다, 3p)
4  인구 구성에 따른 통근 차이        (소섹션 3개: 가~다, 4p)
5  시군별 통근 패턴 분석            (총괄 1p + 시군별 18p = 19p)
```

### 공통 패턴

- 중섹션 수: 5~6개
- 소섹션 수: 중섹션당 2~3개
- 흐름: **현황 파악 → 유형/패턴 분류 → 심화 분석 → 종합/후보 선정**

## 시군별 개별 페이지 (선택적, 상세)

### 총괄 페이지

```
┌─────────────────────────────────┐
│ [경남 전체 지도 — 핵심 지표]      │
│                                 │
│ [그림] 경남 평균 통근거리         │
│ [그림] 경남 광역통근비율 현황      │
└─────────────────────────────────┘
```

- 1~2p, 경남 전체 지도 차트 2개 배치
- 이후 시군별 페이지로 전환됨을 안내하는 역할

### 시군 개별 페이지

```
┌─────────────────────────────────┐
│ ○ 거제시                        │  ← 시군명 (Bold, 14pt)
│                                 │
│ ┌────────────┐ ┌────────────┐   │
│ │ [차트 1]    │ │ [차트 2]    │   │  ← 핵심 차트 2~4개
│ └────────────┘ └────────────┘   │
│ ┌────────────┐ ┌────────────┐   │
│ │ [차트 3]    │ │ [차트 4]    │   │
│ └────────────┘ └────────────┘   │
│                                 │
│ • 불릿 요약 3~5개 (선택)         │  ← 텍스트 해석
└─────────────────────────────────┘
```

| 항목 | 규칙 |
|------|------|
| 제목 | `○ 시군명` (원형 불릿 + 공백 + 시군명) |
| 시군 순서 | 가나다순 18개 |
| 차트 수 | 2~4개 (2x2 격자 배치) |
| 불릿 | 0~5개 (차트가 충분히 설명적이면 생략) |
| 차트 크기 | 페이지 폭의 45% x 2열 |

### 시군 순서 (가나다순, 18개)

```
거제시 → 거창군 → 고성군 → 김해시 → 남해군 →
밀양시 → 사천시 → 산청군 → 양산시 → 의령군 →
진주시 → 창녕군 → 창원시 → 통영시 → 하동군 →
함안군 → 함양군 → 합천군
```

## 체크리스트

- [ ] Ⅳ가 새 페이지에서 시작하는가
- [ ] 중섹션이 5~6개 존재하는가
- [ ] 중섹션 흐름이 "현황→분류→심화→종합" 순서인가
- [ ] 각 소섹션이 "불릿 5~6개 + [그림] 1개 = 1p" 밀도를 준수하는가
- [ ] 모든 불릿에 수치가 1개 이상 포함되었는가
- [ ] [그림]/[표] 캡션이 이미지/표 아래에 배치되었는가
- [ ] 캡션에 번호가 없고 대괄호 접두어만 사용되었는가
- [ ] 시군별 개별 페이지가 가나다순인가 (해당 시)
