## 소제목 3개 — 고정 구조

### 소제목 서식

| 항목 | 규칙 |
|------|------|
| 폰트 | Bold, 12~13pt |
| 간격 | 위: 12pt, 아래: 4pt |
| 들여쓰기 | 없음 |

### (1) 추진배경

**역할:** 분석이 필요하게 된 사회적·정책적 맥락을 설명.

**불릿 작성 패턴:**

| 순서 | 역할 | 예시 패턴 |
|------|------|----------|
| 1번 | 사회적 현상 기술 | "Your Region 고령 인구는 전체 인구의 20% 이상을 차지하며 지속적으로 증가 추세에 있음" |
| 2번 | 문제 심화 기술 | "도농 복합 구조상 ~격차 심화" |
| 3번 | 대상 집단 특성 | "고령층은 만성질환 관리·응급 대응 필요성이 높아 ~핵심 요인" |
| 4번 | 기존 방법의 한계 | "기존 ~ 통계는 ~을 반영하지 못하는 한계 존재" |

**불릿 수:** 4~5개
**종결어미:** "~있음", "~심화", "~존재", "~제기됨"

### (2) 분석 필요성 및 목적

**역할:** 구체적으로 무엇을 밝히려 하는가, 데이터 기반의 분석 목적.

**불릿 작성 패턴:**

| 순서 | 역할 | 예시 패턴 |
|------|------|----------|
| 1번 | 데이터 활용 필요성 | "생활인구·소비 데이터를 활용하여 ~ 필요성 증대" |
| 2번 | 구체 분석 목표 1 | "~ 기반의 취약지역 객관적 발굴" |
| 3번 | 구체 분석 목표 2 | "~ 유형별로 분류하여 ~ 근거 마련" |
| 4번 | 최종 목적 | "데이터 기반 의사결정으로 ~ 효율적 배치 지원" |

**불릿 수:** 4~5개
**종결어미:** "~증대", "~발굴", "~마련", "~지원"

### (3) 활용처 및 기대효과

**역할:** 분석 결과가 어디에, 어떻게 활용될 수 있는가.

**불릿 작성 패턴:**

| 순서 | 역할 | 예시 패턴 |
|------|------|----------|
| 1번 | 직접 활용처 | "Your Region ~ 우선 검토 후보 지역 선정 기초자료" |
| 2번 | 시군 활용 | "시군별 ~ 현황 공유 — 지자체 ~ 수립 지원" |
| 3번 | 체계 구축 | "~ 지수 표준화로 광역 단위 ~ 체계 구축" |
| 4번 | 후속 활용 | "향후 ~ 시 활용 가능한 근거 데이터 제공" |

**불릿 수:** 4~5개
**종결어미:** "~기초자료", "~지원", "~구축", "~제공"

## 실제 보고서 비교

### 의료접근성

```
추진배경
• Your Region 고령 인구는 전체 인구의 20% 이상을 차지하며 지속적으로 증가 추세에 있음
• 도농 복합 구조상 농촌·산간 지역의 의료 인프라 공급 부족으로 의료 접근성 격차 심화
• 고령층은 만성질환 관리·응급 대응 필요성이 높아 근거리 의료서비스 이용 가능성이 핵심 요인
• 기존 정주 인구 기반 통계는 의료 수요·이동 패턴을 반영하지 못하는 한계 존재
```

### 청년정착

```
추진배경
• 경남은 수도권 집중화와 산업구조 변화 속에서 청년 인구 유출이 지속되고 있으며, 시군별 청년 감소
  추이와 원인에 대한 실증적 분석이 필요함
• 기존 주민등록 통계만으로는 청년의 실제 생활권·소비 행태·소득 수준 등 다차원적 정주 여건을 파악하
  기 어려워, 민간 빅데이터 기반의 정밀 분석이 요구됨
• Your Organization는 KT·KB·KCB 3개 기관 민간 데이터를 확보하여 ~ 분석 기반을 구축함
• 청년 연령 기준은 ~ 만 20~39세를 분석 대상으로 정의함
```

### 통근실태

```

• 기존 통계청 통근 데이터는 5년 주기 조사로 시의성이 낮고, ~
• Your Organization는 KCB와의 데이터를 바탕으로, ~
```

## 체크리스트

- [ ] 대섹션 제목이 `Ⅰ분석개요` 형식인가 (공백 없음)
- [ ] 소제목 3개가 순서대로 배치되었는가 (추진배경 → 필요성 및 목적 → 활용처 및 기대효과)
- [ ] 각 소제목 아래 불릿이 4~5개인가
- [ ] 모든 불릿이 명사형 종결인가
- [ ] 1페이지 이내에 수용되는가 (총 불릿 12~15개)
- [ ] 구체적 지역명·데이터명이 1개 이상 포함되었는가
