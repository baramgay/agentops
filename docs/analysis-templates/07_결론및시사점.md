## 불릿 서식

| 항목 | 규칙 |
|------|------|
| 접두어 | `y` (소문자 y) |
| 폰트 | 맑은 고딕, Regular, 10pt |
| 분량 | 1불릿 = 2~4줄 (60~160자) |
| 들여쓰기 | y 접두어 후 공백 2칸, 2줄째 이후 들여쓰기 정렬 |

> 결론 섹션의 불릿은 Ⅳ 분석결과의 `•` 불릿(1~2줄)보다 **길다** (2~4줄).
> 한 불릿에 여러 사실과 해석을 연결하여 종합적으로 기술.

## ▢분석 결과 종합

### 역할

Ⅳ 분석결과의 핵심 수치와 주요 발견사항을 압축 재인용.

### 불릿 작성 패턴

| 순서 | 역할 | 패턴 |
|------|------|------|
| 1번 | 전체 규모·핵심 수치 | "경남 전체 ~ N개이며, ~ N%를 차지" |
| 2번 | 유형/등급 분류 결과 | "~ 유형은 A N개, B N개, C N개로 ~ 가능" |
| 3번 | 유형별 특성 대비 | "도시형 취약은 ~ 구조이고 농촌형 취약은 ~ 구조로 나타남" |
| 4번 | 최상위/최하위 지역 | "최고 ~ 지역은 ~이며 ~로 확인" |
| 5번 | 공간적 패턴 | "~와 ~ 일부는 ~ 축으로 나타났고 ~ 지역은 ~ 축으로 나타남" |
| 6~7번 | 추가 주요 발견 | 분석 주제별 고유 결과 |
| 마지막 | 내부 편차 강조 | "동일 시군 내부에서도 ~ 편차가 ~ 평균만으로는 ~" |

### 실제 사례 (의료접근성)

```
▢분석 결과 종합
y  Your Region 305개 행정동 중 의료접근성 최취약 1등급 지역은 61개로 전체의 20.0를 차지
y  취약 유형은 A유형 25개, B유형 76개, C유형 45개와 AC복합 3개로 확인되어 취약 원인에
   따른 구분이 가능
y  도시형 취약은 고령 수요 집중과 상대적 공급 지연이 결합된 구조이고 농촌형 취약은 의료공
   백과 긴 접근거리가 결합된 구조로 나타남
y  최고 취약점수 지역은 Your Region 양산시 물금읍이며 취약점수는 100.0로 확인
y  양산시와 김해시, 창원권 일부는 대규모 고령 수요가 누적되는 도시형 취약 축으로 나타났고
   서부 내륙 군 지역은 공백 중심의 농촌형 취약 축으로 나타남
y  거제시와 통영시는 C유형 비중이 높아 단순 공백보다 병원급 이상 공급과 중증 대응 측면의
   취약성이 함께 확인
y  행정동 단위 분석 결과는 동일 시군 내부에서도 취약 편차가 크다는 점을 보여주어 시군 평
   균만으로는 실제 우선 대응지를 설명하기 어려움
```

### 불릿 수: 5~8개

## ▢정책적 시사점

### 역할

분석 결과를 기반으로 구체적 정책 방향을 유형별/지역별로 제시.

### 불릿 작성 패턴

| 순서 | 역할 | 패턴 |
|------|------|------|
| 1번 | 유형 A 대응 | "A유형 N개 행정동은 ~처럼 즉시 대응 가능성이 높은 우선 검토 대상" |
| 2번 | 유형 B 대응 | "B유형 지역은 ~ 정책을 병행하는 접근이 필요" |
| 3번 | 유형 C 대응 | "C유형 비중이 높은 ~은 ~ 보완을 함께 검토할 필요" |
| 4번 | 후보지 활용 | "우선 검토 후보지 Top N은 ~ 후보군으로 활용하고 ~을 반영해 최종 대상을 보정할 필요" |
| 5번 | 예외/주의 | "~처럼 ~ 사례는 ~ 주의할 필요" |
| 6번 | 시군 단위 전략 | "취약비율이 높은 시군은 ~ 체계 개편과 권역 협의를 함께 검토할 필요" |
| 7~8번 | 수치 기반 근거 | "N등급 지역의 ~는 N km로 ~ 수준보다 ~" |

### 실제 사례 (통근실태)

```
▢정책적 시사점
y  광역권 교통망 확충이 핵심 과제이며 양산(7.4만 명)·김해(5.4만 명) 중심 부산–양산 BRT·경
   전철, 부산김해경전철 확장 등 광역 대중교통 투자 필요, 동시에 경남 내부 통근축 교통망 개
   선도 중요하며 김해–창원(3.8만 명), 창원–김해(2.3만 명), 창원–함안(2.1만 명), 진주–사천
   (1.2만 명) 등 주요 통근축 중심 도로 및 대중교통 개선 필요
y  소득 계층별 맞춤 정책도 요구되며 저소득층 교통비·주거 지원, 고소득층 장거리 통근 완화를
   위한 경남 내 고소득 일자리 창출, 광역통근 인구가 가장 많은 중소득층 대상 대중교통 확대
   필요
```

### 종결어미 패턴

```
~필요
~과제
~함께 검토할 필요
~보정할 필요
~주의할 필요
~효과적
~바람직
```

### 불릿 수: 5~8개

## ▢향후 과제

### 역할

데이터 한계, 지표 고도화 방향, 후속 분석 계획, 모니터링 체계 등 향후 작업을 제시.

### 불릿 작성 패턴

| 순서 | 역할 | 예시 시작부 |
|------|------|------------|
| 1번 | 데이터 갱신 | "의료기관 현황은 20XX년 기준이므로 ~ 주기적으로 갱신할 필요" |
| 2번 | 지표 고도화 | "종합 취약점수는 ~ 중심으로 산출되었으므로 향후 ~ 추가 반영할 필요" |
| 3번 | 지표 확장 | "향후 ~ 등 ~ 함께 결합해 ~ 고도화할 필요" |
| 4번 | 유형/기준 보완 | "B유형은 ~ 구분하기 위한 현장 점검과 보완 자료 확보 필요" |
| 5번 | 시계열 분석 | "20XX년부터 ~ 연도별 취약 변화 추적을 확대해 ~ 시계열 분석 체계 구축 필요" |
| 6번 | 정책 효과 평가 | "~ 배치 이후 ~ 변화를 추적해 정책 효과성을 평가할 수 있는 방법론 마련 필요" |
| 7번 | 실행 여건 검토 | "행정동 단위 결과를 실제 사업으로 연결하기 위해서는 ~ 집행 여건을 함께 검토할 필요" |
| 8번 | 정밀도 향상 | "도서·산간 지역은 ~ 향후 이동시간 기반 접근성 분석으로 정밀도를 높일 필요" |
| 9번 | 타 분야 연계 | "복지와 교통 등 타 분야 데이터와 연계한 ~ 확장 가능성을 검토할 필요" |
| 10번 | 현장 검증 | "최종 정책 집행 전에는 ~ 현장 검증을 실시해 ~ 확인할 필요" |

### 실제 사례 (청년정착)

```
▢향후 과제
y  본 분석의 5개 정착잠재지수 구성지표를 핵심 모니터링 KPI로 설정하고, KT·KB·KCB 민간
   빅데이터 3종의 연간 갱신 주기에 맞춰 지수를 재산출함으로써 시군별 순위 변화를 추적하고
   정책 개입의 적시성을 확보하는 체계 구축이 우선 과제이며...
y  향후 분석에서는 본 연구에서 단년도로 제한된 KCB 소득·신용 지표의 시계열 확장이 선결
   과제이며, 청년 연령 내에서 20대와 30대 간 유출·유입 패턴의 세대 내 차이를 세분화하는
   심층 분석이 필요...
```

### 종결어미 패턴

```
~필요
~과제
~구축 필요
~마련 필요
~높일 필요
~검토할 필요
```

## 체크리스트

- [ ] 대섹션 제목이 `Ⅴ결론 및 시사점` 형식인가
- [ ] 하위 블록 3개(분석 결과 종합, 정책적 시사점, 향후 과제)가 모두 존재하는가
- [ ] 하위 블록 제목에 `▢` 접두어가 사용되었는가
- [ ] 불릿 접두어가 `y`인가 (`•`가 아님)
- [ ] 분석 결과 종합에 Ⅳ의 핵심 수치가 재인용되었는가
- [ ] 정책적 시사점이 유형별/지역별로 구체적인가
- [ ] 향후 과제에 데이터 한계와 후속 분석 방향이 포함되었는가
- [ ] 각 불릿이 2~4줄 분량인가 (Ⅳ 불릿보다 길어야 함)
- [ ] 2~3페이지 이내에 수용되는가
