## (1) 요구사항

**역할:** 이 분석이 충족해야 할 기능적 요건을 나열.

### 불릿 작성 패턴

| 순서 | 역할 | 패턴 |
|------|------|------|
| 1~2번 | 핵심 지표 산출 요건 | "~ 데이터 기반 ~별 ~ 산출" |
| 3번 | 보조 지표 활용 | "~ 데이터를 보조지표로 활용하여 ~" |
| 4번 | 분류 체계 요건 | "~ 유형별 분류 체계 수립: ~으로 세분화하여 ~" |
| 5~6번 | 시각화/공간 분석 요건 | "GIS 코로플레스 지도를 통한 ~" |

### 실제 사례

**의료접근성:**
```
• 고령층 의료 수요 측정: KT 생활인구 데이터 기반 행정동별 60세 이상 평균 생활인구 산출
• 의료 공급 측정: 심평원 의료기관 현황을 활용한 가중 의료공급 지수 산출
• KB 의료소비 데이터를 보조지표로 활용하여 실제 의료 이용 현황 파악
• 취약 유형별 분류 체계 수립: 도시형·농촌·중증공급 취약으로 세분화하여 정책 활용성 제고
```

**청년정착:**
```
• 경남 18개 시군 청년(20~39세) 등록인구의 2018~2025년 시계열 변화 및 시군 간 증감률 비교
• KT 이동인구 OD 데이터를 활용하여 경남 시군별 타지역 청년 순유입·순유출 현황 분석
• KB카드 매출 데이터로 시군별 청년 1인당 카드 소비액 및 소비 특성 분석
• KCB 신용융합 데이터로 시군별 청년 1인당 추산소득·신용점수·소득분위 분포 분석
• 위 지표들을 z-score 표준화 후 가중합산한 '청년 정착잠재지수' 산출 및 시군 순위화
• GIS 코로플레스 지도를 통한 지표별 공간 분포 시각화
```

**통근실태:**
```
• 경남 18개 시군의 통근 패턴(거주지-근무지 이동)을 정량적으로 분석하여, 시군별 평균 통근거리,
  자족도, 광역통근 비율 등 핵심 지표를 산출함
• 부산·울산 광역권으로의 통근 현황을 파악하고, ~ 근거를 마련함
• 경남 내 시군 간 통근 패턴을 분석하여, 주요 통근축을 파악하고 ~ 방향을 제시함
• 소득 구간별 통근 패턴 차이를 분석하여, ~ 근거를 도출함
• 성별 및 외국인 근로자의 통근 패턴을 분석하여, ~ 방향을 제시함
• 시군별 통근 패턴을 4가지 유형으로 분류하고, ~ 기반을 마련함
```

## (2) 분석방법

**역할:** 구체적인 분석 기법, 지표 산출식, 등급/유형 분류 기준을 명시.

### 불릿 작성 패턴

| 순서 | 역할 | 패턴 |
|------|------|------|
| 1번 | 분석 대상·기간 | "분석 대상: 경남 18개 시군 ~" |
| 2번 | 핵심 지표 산출식 | "수요 지표: ~ = 가중치 × 변수" |
| 3번 | 점수/등급 산출 | "취약점수: ~ z-score ~ 0~100점 정규화" |
| 4번 | 등급 구분 기준 | "등급 구분: 1등급(상위 20%) ~ 5분위 분류" |
| 5번 | 유형 분류 기준 | "유형 분류: 수요·공급 특성에 따라 A·B·C 3유형" |
| 6번 | 공간 분석 | "공간 분석: GeoPandas 기반 ~ 지도 생성" |

### 지표 산출식 표기 규칙

수식은 불릿 안에 인라인으로 기술한다:

```
• 공급 지표: 가중 의료공급 지수 = 종합병원×10 + 병원×3 + 의원×1 + 한의원×0.5 + 치과×0.3
• 취약점수: 수요(z-score)와 1인당 의료소비 역수(z-score)의 합산 후 0~100점 정규화
• 지수 산출: 5개 지표 z-score 표준화 후 동일 가중치(각 20%) 합산
```

- 수식 기호: `×`, `+`, `-`, `/` (유니코드 곱셈 기호 사용)
- 괄호: 보조 설명에 사용 "(z-score)", "(KT, 2025년 연간)"
- 가중치: 구체적 수치 명시
