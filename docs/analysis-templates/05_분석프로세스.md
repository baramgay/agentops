| 항목 | 규칙 |
|------|------|
| 표기 | `Ⅲ분석 프로세스` (로마 숫자 + 공백 + 제목) |
| 폰트 | Bold, 16~18pt |
| 서식 | Ⅰ, Ⅱ와 동일한 컬러바/배경 스타일 |

## 1. 데이터 수집 및 전처리

### 가. 데이터 수집 배경

**역할:** 왜 이 데이터들을 선택했는가, 데이터의 강점.

**불릿 패턴:**

| 순서 | 패턴 | 예시 |
|------|------|------|
| 1번 | 분석 범위 설정 | "Your Region 18개 시군 305개 행정동 단위로 분석 범위 설정" |
| 2번 | 데이터 A 강점 | "KT 생활인구는 ~ 실제 체류 인구 파악에 적합" |
| 3번 | 데이터 B 강점 | "KB 의료소비는 ~ 실제 의료 이용 금액·건수 확인 가능" |
| 4번 | 데이터 C 강점 | "심평원 의료기관 데이터는 ~ 상세 위치 정보 포함" |

**불릿 수:** 3~4개
**종결어미:** "~적합", "~가능", "~포함"

### 나. 데이터 생성 과정 (또는 "데이터 수집 과정")

**역할:** 데이터 구조, 버전 차이, 수집 경위 등 기술적 세부사항.

**불릿 패턴:**

| 순서 | 패턴 | 예시 |
|------|------|------|
| 1번 | 데이터 A 수집/변환 | "KT 생활인구: 2022년부터 50m 격자 단위 제공 시작 → 시군 단위 집계로 통합" |
| 2번 | 데이터 B 수집/변환 | "KT 이동인구: 경남→타시도(유출) / 타시도→경남(유입) OD 파일 분리 수집" |
| 3번 | 데이터 C 보정 | "KB카드: ms 보정계수 적용 후 연속 시계열 구축" |
| 4번 | 데이터 D 기준 | "KCB: 최신 폴더(2025.11 기준) 단일 사용" |
| 5번 | 외부 데이터 | "주민등록: KOSIS에서 5세 단위 연령별 인구 수집" |

**불릿 수:** 4~6개 (데이터 수에 비례)
**종결어미:** "~통합", "~수집", "~구축", "~사용"

### 다. 데이터 전처리 과정

**역할:** 코드 통일, 결측 처리, 보정, 변환 등 구체적 전처리 작업.

**불릿 패턴:**

| 순서 | 패턴 | 예시 |
|------|------|------|
| 1번 | 데이터 통합/정리 | "KT 신구버전 통합: ~ 컬럼 통일 병합" |
| 2번 | 코드 표준화 | "창원시 통합: 마산·진해 합산 이전 코드 → 창원시(48110)로 통일" |
| 3번 | 보정 처리 | "KB ms 보정: 2018~2022 통합본에 보정계수(~0.145) 나눗셈 적용" |
| 4번 | 결측치 처리 | "결측치 처리: 의료소비 0원 지역은 실제 부재로 처리(결측 아님)" |
| 5번 | 단위 변환 | "평균 출퇴근거리(m 단위)를 km 단위로 변환" |
| 6번 | 변수 생성 | "통근거리를 5개 구간으로 분류하여 분석 변수 생성" |

**불릿 수:** 4~6개
**종결어미:** "~병합", "~통일", "~적용", "~처리", "~생성"

**핵심 규칙:**
- 구체적 코드·계수·기준값을 명시 (예: "48110", "~0.145", "0~5km")
- 기술적 의사결정 근거를 괄호로 보충 (예: "(결측 아님)")

## 2. 데이터 분석 및 시각화

### 가. 분석 단계 (또는 "분석 체계")

**역할:** 분석 절차를 단계별로 나열.

**작성 패턴:**

```
• 1단계: [데이터 수준의 기초 작업] (예: "행정동별 수요·공급 지표 산출 및 표준화")
• 2단계: [핵심 지표/점수 산출] (예: "취약점수 계산 및 0~100점 정규화")
• 3단계: [등급/분위 부여] (예: "취약점수 5분위 등급 부여")
• 4단계: [유형 분류] (예: "유형 분류 및 복합유형 식별")
• 5단계: [후보 선정/순위화] (예: "유형별 1차 후보군 지역 선정")
• 6단계: [공간 분석] (예: "GIS 공간 분포 시각화")
• 7단계: [종합] (예: "종합 지수 산출 및 유형화")
```

**불릿 수:** 5~7개 (단계 수에 따라)
**표기:** `N단계:` 접두어 필수

### 나. 시각화 방법 (및 도구)

**역할:** 사용한 차트 유형과 용도 매핑.

**작성 패턴:**

```
• 단계구분도(Choropleth Map): 행정동 경계 기반 ~ 공간 시각화
• 산포도(Scatter Plot): 수요·공급 축 기반 유형 구분 시각화
• 롤리팝 차트: 시군별 ~ 비율 비교
• 시계열 추이: 선 그래프 (연도별 ~ 변화)
• 시군별 비교: 수평 막대 그래프 (18개 시군 지표 비교, 경남 평균선 표시)
• 다차원 종합: 버블 차트 (x: ~, y: ~, 크기: ~)
```

**불릿 수:** 3~6개
**표기:** `차트유형(영문명): 용도` 형식

### 색상 규칙 (언급 시)

```
• 색상 규칙: 증가·유입·양호는 녹색, 감소·유출·위험은 적색
• 각 시각화에 평균선(빨간 점선) 표시, 주요 수치 라벨 명시
```

### 다. 분석 도구 (선택적)

청년정착·통근 보고서에서 확인된 패턴:

```
• Python 3.x 기반 데이터 처리 및 시각화
• 주요 라이브러리: Pandas(데이터 처리), Matplotlib(시각화), Seaborn(통계 시각화),
  GeoPandas(공간 분석), Openpyxl(엑셀 처리)
• Streamlit 기반 인터랙티브 대시보드 구축
```

**불릿 수:** 2~3개
**종결어미:** "~처리", "~시각화", "~구축"
