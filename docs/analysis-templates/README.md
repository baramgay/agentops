## 학습 표본 PDF 4종

실제 보고서 예시:
- 고령층 의료접근성 취약지 발굴 분석 (28p)
- 경남 청년 정착 잠재지역 발굴 분석 (41p)
- 경남 제조업특화시군 산업활력 진단 분석 (28p)
- 경남 도민 정책 수요 발굴 분석 (41p)

표본 PDF 4종은 본 레포의 `docs/analysis-templates/samples/` 에 보관:

- `고령층의료접근성취약지발굴분석(변경양식).pdf` (28p, ~5.4MB)
- `경남청년정착잠재지역발굴분석(변경양식).pdf` (41p, ~5.5MB)
- `경남제조업특화시군산업활력진단분석(변경양식).pdf` (28p, ~3.0MB)
- `Your Region민정책수요발굴분석(변경양식).pdf` (41p, ~5.1MB)

reporter 에이전트는 다른 PC에서도 동일한 표본을 참조할 수 있다.

## 한자 사용 금지

본 폴더 내 모든 파일은 순한글로 작성. "분석"은 한자(U+6790) 사용 금지, 순한글(U+BD84+U+C11D)만 사용.
