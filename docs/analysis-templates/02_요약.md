# [2] 요약 — 상세 규칙

> 보고서의 두 번째 페이지. 분석 전체를 1페이지로 압축한 핵심 요약.

---

## 레이아웃

```
┌─────────────────────────────────┐
│ 01                              │  ← 과제 일련번호 (좌상단)
│ [분석 주제명 전체 — 1줄]         │  ← 주제명 (좌측 정렬)
│ 자체발굴                         │  ← 과제 구분 (우측 정렬)
│ Your Organization 팀장 이 현 기    │  ← 작성자 (우측 정렬)
│ ─────────────────────────────── │  ← 구분선 (선택)
│ 요  약                          │  ← 소제목 (중앙 또는 좌측)
│                                 │
│ [요약 본문 — 3~4문단]            │
│                                 │
│                                 │
└─────────────────────────────────┘
```

---

## 구성 요소 상세

### 헤더 영역 (상단 20%)

| 요소 | 위치 | 폰트 | 비고 |
|------|------|------|------|
| 과제 일련번호 "01" | 좌상단 | Bold, 18~24pt | 표지와 동일 번호 |
| 주제명 전체 | 일련번호 아래 | Bold, 14~16pt | 1줄로 표기 (줄바꿈 없음) |
| 과제 구분 | 우측 정렬 | Regular, 10pt | "자체발굴" / "수요대응" / "정기분석" |
| 작성자 | 과제 구분 아래 | Regular, 10pt | "Your Organization 팀장 이 현 기" |

과제 구분과 작성자는 주제명 오른쪽 또는 아래에 우측 정렬로 배치한다.

### "요  약" 소제목

| 항목 | 규칙 |
|------|------|
| 표기 | "요  약" (요와 약 사이 공백 2칸) |
| 폰트 | Bold, 14~16pt |
| 위치 | 헤더 영역 아래, 좌측 정렬 또는 중앙 정렬 |
| 간격 | 위: 12pt, 아래: 6pt |

### 요약 본문

| 항목 | 규칙 |
|------|------|
| 총 분량 | 200~400자 (A4 1p 내 수용) |
| 문단 수 | 3~4개 문단 |
| 폰트 | 맑은 고딕, Regular, 10~11pt |
| 줄간격 | 1.5배 |
| 들여쓰기 | 없음 (문단 간 1줄 간격으로 구분) |

## 요약 본문 작성 구조

### 문단별 역할

| 문단 | 역할 | 분량 | 종결어미 |
|------|------|------|---------|
| 1문단 | 분석 대상·범위·방법 요약 | 50~80자 | "~수행되었다" |
| 2문단 | 핵심 분석 결과 (수치 중심) | 80~150자 | "~확인되었다", "~나타났다" |
| 3문단 | 유형/등급 분류 결과 | 60~100자 | "~구분된다", "~분류되었다" |
| 4문단 | 정책 방향·활용 가능성 | 50~80자 | "~활용할 수 있다" |

### 필수 포함 요소

- [ ] 분석 대상 지역 (예: "Your Region 18개 시군 305개 행정동")
- [ ] 분석 대상 인구/기간 (예: "고령층(60세 이상)", "2018~2025년")
- [ ] 핵심 결과 수치 1~3개 (예: "61개 행정동이 1등급으로 분류")
- [ ] 유형 분류 결과 (예: "A유형 25개, B유형 76개, C유형 45개")
- [ ] 활용 데이터 출처 1~2개 (예: "KT 생활인구, KB 의료소비")

## 실제 보고서 사례

### 의료접근성 보고서 요약 (발췌)

```
Your Region 18개 시군 305개 행정동을 대상으로 고령층(60세 이상) 의료접근성
취약지역을 분석한 결과, 취약점수 상위 20%에 해당하는 61개 행정동이 1등급
(최취약)으로 분류되었다. 취약유형은 도시형 수요과밀(A) 25개, 농촌 의료공백
(B) 76개, 중증공급 취약(C) 45개와 AC복합 3개로 구분된다.
...
```

### 청년정착 보고서 요약 (발췌)

```
본 분석은 KT 통신 생활·이동인구, KB국민카드 매출, KCB 신용융합 데이터 등
민간 빅데이터 6종을 활용하여 경남 18개 시군 청년(만 20~39세)의 인구 유출
실태와 정착 가능성을 2018~2025년 장기 시계열로 종합 진단하고...
```

### 통근실태 보고서 요약 (발췌)

```
본 분석은 KCB 신용정보 통근통계 데이터를 활용하여 경남 18개 시군 약 182만
명의 통근 패턴을 분석하고, 평균 통근거리·자족도·광역통근 비율 등 핵심 지표를
도출하여 교통·일자리 정책 수립의 근거를 마련하기 위해 수행되었다.
```

## 문체 패턴

### 허용되는 종결어미

```
~분석한 결과
~으로 분류되었다
~으로 구분된다
~으로 나타났다
~으로 확인되었다
~위해 수행되었다
~활용할 수 있다
~마련하기 위해 수행되었다
~필요가 있다
```

### 금지 패턴

```
❌ "~입니다" (경어체)
❌ "~했습니다" (경어체)
❌ "~인 것으로 보인다" (모호한 추정)
❌ "~라고 할 수 있겠다" (불필요한 겸양)
```

## 체크리스트

- [ ] 과제 일련번호가 표지와 동일한가
- [ ] 주제명이 1줄로 표기되었는가 (줄바꿈 없음)
- [ ] 과제 구분("자체발굴" 등)이 표기되었는가
- [ ] 작성자 소속·직급·이름이 모두 포함되었는가
- [ ] "요  약" 소제목의 공백이 2칸인가
- [ ] 요약 본문이 200~400자인가
- [ ] 구체적 수치가 1개 이상 포함되었는가
- [ ] 과거형 서술체("~되었다")로 작성되었는가
- [ ] 1페이지 이내에 수용되는가
