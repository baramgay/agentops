## 구성 요소

### 과제 일련번호

| 항목 | 규칙 |
|------|------|
| 위치 | 주제명 바로 위, 수평 중앙 |
| 표기 | 2자리 숫자 (01, 02, 03...) |
| 폰트 | 맑은 고딕, Bold, 36~48pt |
| 색상 | 진한 회색(#333333) 또는 기관 CI 메인 컬러 |

### 주제명

| 항목 | 규칙 |
|------|------|
| 위치 | 페이지 수직 중앙~약간 위 (상단 40% 지점) |
| 정렬 | 수평 중앙 정렬 |
| 폰트 | 맑은 고딕, Bold, 28~36pt |
| 줄 수 | 2~3줄 (의미 단위로 줄바꿈) |
| 줄간격 | 1.5배 |
| 색상 | 검정(#000000) 또는 진한 남색 |

### 줄바꿈 기준

주제명이 길면 의미 단위로 자른다:

```
✅ 올바른 예시:
  고령층 의료접근성
  취약지 발굴 분석

  경남 청년 정착
  잠재지역 발굴 분석

  2025년 2분기
  빅데이터 기반
  통근실태 분석

❌ 잘못된 예시:
  고령층 의료접근성 취약지
  발굴 분석                    ← "취약지 발굴"이 분리됨
```

- 한 줄에 10~14자가 적정
- 조사("의", "을", "에") 앞에서 자르지 않음
- "분석"은 항상 마지막 줄 끝에 위치

## 배경 처리

| 유형 | 적용 방법 |
|------|----------|
| 단색 배경 | 연한 회색(#F5F5F5) 또는 흰색 |
| 기관 CI | Your Region CI 컬러 적용 가능 (좌측 컬러바, 하단 로고 등) |
| 이미지 배경 | 사용하지 않음 (깔끔한 단색 유지) |

## python-docx 구현 가이드

```
- 페이지 방향: A4 세로 (210mm x 297mm)
- 여백: 상하좌우 2cm
- 과제번호: add_paragraph → run.bold = True, run.font.size = Pt(40)
- 주제명: add_paragraph → run.bold = True, run.font.size = Pt(30)
- 정렬: paragraph.alignment = WD_ALIGN_PARAGRAPH.CENTER
- 상단 여백 확보: 빈 paragraph + paragraph.space_before = Cm(8)
- 페이지 나누기: 표지 끝에 add_page_break()
```
