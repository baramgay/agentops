## 아키텍처 요약

```
src/nuristat/
  core/           — Dataset, VariableMeta, enums, validation
  io/             — CSV/Excel/SPSS 가져오기·내보내기, project_store (.swb = ZIP+Parquet)
  analysis/       — 30+ 분석 모듈, registry (lazy import), visualization
  ui/
    models/       — SPSSGridModel (가상그리드·스냅샷 undo·dtype캐싱)
    delegates/    — CellDelegate (값라벨 QComboBox)
    dialogs/      — 분석 다이얼로그, _dialog_helpers.py (display_label·populate_list_widget)
    main_window.py — 메뉴·상태바·에이전트 라우팅
    data_view.py   — Formula Bar, 셀 편집, 선택 상태바
    variable_view.py — 속성 일괄 복붙
    syntax_editor.py — 구문 실행 (T-TEST·RECODE·SELECT IF)
  syntax/         — 구문 로거
tests/            — 4,714+ passed (v3.6.0, 2026-06-09 기준, 신규 23건 포함)
```

```powershell
# Inno Setup 컴파일
& "C:\Users\username\AppData\Local\Programs\Inno Setup 6\ISCC.exe" NuriStat.iss
```
