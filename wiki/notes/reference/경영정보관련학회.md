# Metis Data Pipeline Platform(DPP) 구축 사례

## 개요

Metis DPP(Data Pipeline Platform)는 데이터 수집부터 품질관리, 가공, 표준화, AI 활용까지 전 과정을 통합 관리하는 데이터 파이프라인 플랫폼이다.

주요 목적은 다음과 같다.

- 데이터 수집 자동화
- 데이터 품질 관리 체계 구축
- 데이터 가공 프로세스 자동화
- 데이터 자산화
- AI 기반 지능형 서비스 구현

---

# 구축 사례

## 1. 기관공유데이터 관리시스템 구축

### 목적

기관별 산재된 데이터를 통합 관리하고 데이터 활용성을 높이기 위한 플랫폼 구축

### 주요 기능

- 데이터 수집
- 표준관리
- 데이터 가공
- 품질관리
- 데이터 현황 모니터링
- 공유·개방 데이터 생성

### 기대효과

- 데이터 관리 효율성 향상
- 데이터 품질 확보
- 기관 내 데이터 자산화 기반 마련

---

## 2. 국가공유데이터 플랫폼 구축

### 목적

범정부 데이터 공유 체계 구축 및 데이터 활용 활성화

### 주요 기능

- AI 기반 검색
- 데이터 검색/분석/제공
- AI 서비스 연계
- 데이터 자산 관리

### 기대효과

- 데이터 접근성 향상
- 데이터 활용도 증대
- AI 활용 기반 확보

# 핵심 메시지

## DPP

데이터 수집부터 품질관리, 가공, 표준화까지 자동화하여 데이터 활용성을 높이는 플랫폼

## AIP

데이터 자산화를 넘어 지식그래프와 생성형 AI를 활용한 데이터 지능화 플랫폼

## 최종 목표

```text
데이터 수집
→ 데이터 품질관리
→ 데이터 자산화
→ 지식그래프 구축
→ 생성형 AI 활용
→ 의사결정 지원
```

---

# 개인 메모

- 데이터 거버넌스와 데이터 품질관리의 중요성 강조
- 데이터 파이프라인 자동화가 핵심 가치
- 지식그래프 + RAG 구조로 AI 서비스 확장
- 데이터 자산화 이후 AI 지능화 단계로 발전
- 공공기관 데이터 통합 및 활용 사례 참고 가능
