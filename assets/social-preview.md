# Social Preview Assets

This folder contains visual and descriptive assets for social media sharing.

## GitHub Social Preview

**Recommended image**: `dashboard-preview.png`

Upload `dashboard-preview.png` as the GitHub repository social preview image:
- Settings → General → Social preview → Edit → Upload image

**Dimensions**: 1280 × 640 px (GitHub recommended)

## Open Graph / Twitter Card

Use the following meta description for link previews:

> **agentops** — A structured multi-agent workflow system built on plain markdown files and Python scripts. Define agent roles, manage a knowledge wiki, and track status with a local real-time dashboard.

**Short tagline**: "Structured AI agents. Plain files. No black boxes."
