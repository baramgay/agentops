KT 통신 데이터(OD 이동인구, 생활인구)는 특정 시간대 기지국 관측 기반의 **일별 스냅샷** 집계값이다.

**Why:** 동일 인물이 매일 반복 관측되면 매일 1씩 누적되므로, "체류인일(人日)"은 실제 이동하거나 거주한 사람 수(명)와 직접 비교할 수 없다. 시군 간 이동 방향·추이 분석에는 적합하지만, 누적 체류인일을 인구 수로 해석하면 오해 발생.

**How to apply:**
- 보고서에서 KT OD 수치를 표시할 때 반드시 "체류인일(기지국 관측 스냅샷 기반)" 명시
- 절대 수치(누적 인일)보다는 순유입률(%) 또는 백만 체류인일 단위로 표현해 오해 방지
- "몇 명이 이동" 표현 금지, "체류인일 기준 순유입" 표현 사용
- 누적 수치 제시 시 "실제 이동 인구 수와 다름" 각주 필수
