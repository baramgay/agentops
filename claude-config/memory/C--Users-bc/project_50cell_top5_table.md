50cell 격자 분포 차트(16_01~16_06)에서 표시된 밀도 상위 5개 구역(500m 반경)에 대해 표 형식으로 보고서에 설명 추가 예정.

**Why:** 별(★) 표시만으로는 어느 지역인지 파악이 어려우므로, 각 구역의 위치(읍면동명), 청년 생활인구 밀도 등을 표로 정리해 보고서 본문에 삽입.

**How to apply:** 추후 보고서 수정 작업 시, 시군별 격자 분포 차트 하단 또는 인근에 "상위 5개 밀집 구역" 표 삽입. pick_spread_zones() 결과에서 각 구역의 좌표로 읍면동명 역지오코딩하여 표 생성.
