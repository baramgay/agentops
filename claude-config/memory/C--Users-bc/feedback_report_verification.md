**Why:** 2026-03-19 세션에서 여러 차례 보고서 수정 오류 발생. 직관적 판단이나 기억에 의존한 수정은 새로운 오류를 만들어냄.

**How to apply:**
- 수치 또는 순위를 수정하기 전에 해당 근거 CSV 파일을 직접 확인할 것
- 보고서 수정 후 verify_report.py 실행으로 96개 항목 자동 검증 수행
- 스크립트: `C:\Users\bc\verify_report.py`
