'분석'을 쓸 때 한자 '석'를 혼용하지 말고 항상 순한글 '분석'으로 표기할 것. 2026-03-23 재확인: "앞으로 절대 분석 한자로 쓰지마" 지시.

**Why:** 사용자가 명시적으로 여러 차례 요청. 파일명, 경로, 텍스트, 코드 주석 등 어디서든 '分석', '분석', '分석' 등 한자 포함 표기 금지.

**How to apply:** 코드, 경로, 설명, 파일명 생성 등 모든 상황에서 오직 순한글 '분석' 사용. 폴더/파일명 작성 시 한자 문자(U+4E00~U+9FFF) 포함 여부를 항상 확인. 2026-04-07: CLAUDE.md 광역 지침에도 추가함.
