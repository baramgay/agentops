---
name: sequential-thinking 사용 규칙
description: 시각화 검토 또는 분석 결과 검토 시 sequential-thinking MCP를 항상 사용
type: feedback
---

시각화(차트, 그래프 등) 또는 분석 결과를 검토할 때는 항상 sequential-thinking MCP 도구를 사용한다.

**Why:** 사용자가 시각화/분석 결과 검토 시 단계적 사고를 통해 더 체계적이고 누락 없는 검토를 원함.

**How to apply:** 차트 생성 결과 확인, 분석 수치 검토, 보고서 내용 검증 등 시각화 또는 분석 결과를 평가하는 모든 상황에서 sequential-thinking MCP의 sequentialthinking 도구를 호출하여 단계적으로 검토한다.
